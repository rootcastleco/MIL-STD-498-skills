# Release Notes: v0.1.0

Date: 2026-06-19

## Summary

Initial public source package for the MIL-STD-498 ChatGPT Skill.

## Contents

- Packaged skill archive: `skill.zip`
- Extracted skill source: `mil-std-498/`
- Skill entrypoint: `mil-std-498/SKILL.md`
- Utility scripts:
  - `mil-std-498/scripts/mil498_bootstrap.py`
  - `mil-std-498/scripts/validate_mil498_docs.py`
- Reference material:
  - DID catalog
  - document structures
  - tailoring and CDRL guidance
  - quality checklist
  - source index
- Example documentation set:
  - `examples/telemetry-gateway/SRS.md`
  - `examples/telemetry-gateway/SDD.md`
  - `examples/telemetry-gateway/STD.md`
  - `examples/telemetry-gateway/STR.md`
  - `examples/telemetry-gateway/traceability.csv`

## Validation

The repository includes `.github/workflows/validate-skill.yml`, which checks:

1. `skill.zip` exists and can be inspected with `unzip -t`.
2. Utility scripts compile under Python 3.11.
3. A sample document set can be generated.
4. The generated document set passes the bundled validator.

## Compliance note

This release supports MIL-STD-498-style documentation workflows. It does not assert contractual compliance by itself. Formal use must be bound to the controlling SOW, CDRL, DID revision, and tailoring decisions.

## Known limitations

- The repository does not bundle upstream MIL-STD-498 PDF contents.
- GitHub Release/tag publication should be created from this release note and `skill.zip` artifact when release tooling is available.
