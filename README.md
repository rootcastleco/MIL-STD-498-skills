# MIL-STD-498 Skills

This repository contains a packaged MIL-STD-498 skill set and the extracted working files.

## Repository Structure

- `skill.zip` — original packaged skill archive.
- `mil-std-498/SKILL.md` — main skill definition.
- `mil-std-498/scripts/` — utility scripts for bootstrapping and validation.
- `mil-std-498/references/` — reference guides and supporting documentation.
- `mil-std-498/agents/` — agent configuration files.

## Included Components

The extracted skill includes:

- Skill specification and usage guidance
- Bootstrap script for setting up MIL-STD-498 document workflows
- Validation script for checking documentation consistency
- Reference materials for DIDs, document structures, tailoring, and quality checks

## Usage

1. Review `mil-std-498/SKILL.md` to understand the skill behavior.
2. Use the scripts in `mil-std-498/scripts/` for setup and validation tasks.
3. Consult the `mil-std-498/references/` documents while preparing deliverables.
